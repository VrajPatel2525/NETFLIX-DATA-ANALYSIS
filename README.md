# Netflix Data Analysis

## Project Overview

This project analyzes the Netflix Titles dataset using Python. The objective is to perform data cleaning, exploratory data analysis (EDA), and data visualization to gain meaningful insights into Netflix's content.

## Dataset

* Dataset: netflix_titles.csv
* Source: Kaggle

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## Features

* Data Loading
* Data Cleaning
* Movies vs TV Shows Analysis
* Top 10 Countries
* Content Added Over the Years
* Top 10 Genres
* Ratings Distribution
* Movie Duration Analysis

## How to Run

1. Install the required libraries:
   pip install -r requirements.txt

2. Run the Python file:
   python Netflix_Data_Analysis.py

## Key Insights

* Netflix contains more Movies than TV Shows.
* The United States has the highest number of Netflix titles.
* Netflix content increased rapidly after 2015.
* Drama and International content are among the most common genres.
* TV-MA is one of the most frequent content ratings.
